## Changes Made

### `src/app/r/[slug]/page.tsx`
- Added `alternates.canonical` in `generateMetadata`
- Added OG image reference (`/opengraph-image`)
- Changed `twitter:card` from `summary` → `summary_large_image`
- Added full `schema.org/Recipe` JSON-LD structured data block including:
  - `recipeIngredient`, `recipeInstructions` (as `HowToStep`)
  - `prepTime`, `cookTime`, `totalTime` (ISO 8601 duration format)
  - `recipeCuisine`, `keywords` (from tags)
  - `nutrition` block (when present)
  - `author` as Organization

### `src/app/page.tsx`
- Added `export const metadata` with `alternates.canonical`
- Added `WebApplication` JSON-LD structured data (schema.org) with name, description, applicationCategory, offers

### `src/app/robots.ts`
- Changed sitemap URL from `NEXT_PUBLIC_APP_URL` → `NEXT_PUBLIC_SITE_URL` (consistent with sitemap.ts)

### `src/app/sitemap.ts`
- Removed `/kitchen` (auth-gated, would waste crawl budget on a redirect)
- Added dynamic DB query to include all public recipe pages (`/r/[slug]`) with `updatedAt` timestamps
- Added `export const dynamic = 'force-dynamic'` to prevent build-time DB query failure
- Converted to `async` function returning `Promise<MetadataRoute.Sitemap>`

### `src/app/privacy/page.tsx`
- Added `alternates.canonical`
- Added `openGraph` block (title, description, siteName)

### `src/app/terms/page.tsx`
- Added `alternates.canonical`
- Added `openGraph` block (title, description, siteName)

### `src/app/(app)/loading.tsx` (new)
- Added route-level loading skeleton for all authenticated app pages
